# 墨染剧本杀网站公网部署指南

## 需求分析
网站：静态HTML页面（quick-site.html）
当前：仅局域网访问（192.168.1.2:8899）
目标：让外网用户也能访问

---

## 方案一：ngrok内网穿透（免费，推荐）

### 操作步骤：
1. 注册ngrok账号：https://dashboard.ngrok.com/signup
2. 获取authtoken：https://dashboard.ngrok.com/get-started/your-authtoken
3. 下载ngrok：https://ngrok.com/download
4. 将ngrok.exe放到网站目录
5. 编辑ngrok-config.yml，填入你的authtoken
6. 双击运行start-ngrok.bat

### 优点：
- 免费（每月1GB流量）
- 快速简单
- 自动分配域名（如：moran-script.ngrok-free.app）
- 支持HTTPS

### 访问地址：
格式：https://[随机域名].ngrok-free.app/quick-site.html
示例：https://moran-script.ngrok-free.app/quick-site.html

---

## 方案二：花生壳（国内稳定）

### 操作步骤：
1. 访问：https://hsk.oray.com/download/
2. 下载Windows版花生壳客户端
3. 注册账号（有免费版）
4. 安装并登录花生壳
5. 添加内网映射：
   - 应用名称：墨染剧本杀网站
   - 内网主机：192.168.1.2
   - 内网端口：8899
   - 外网端口：80或自定义
   - 外网域名：免费子域名（如：yourname.oicp.net）

### 优点：
- 国内网络优化
- 稳定性较好
- 有免费版

## 方案三：Vercel/GitHub Pages（永久部署）

### 操作步骤（Vercel）：

#### 1. 准备工作：
```bash
# 创建项目结构
mkdir moran-website
cd moran-website

# 复制网站文件
copy C:\Users\admin\WorkBuddy\20260401001834\quick-site.html index.html
copy C:\Users\admin\WorkBuddy\20260401001834\web-pdfs\ web-pdfs\

# 创建GitHub仓库
git init
git add .
git commit -m "墨染剧本杀网站上线"
git branch -M main
```

#### 2. 部署到Vercel：
1. 注册Vercel：https://vercel.com/signup
2. 导入GitHub仓库
3. 自动构建部署
4. 访问地址：proj-name.vercel.app

#### 3. 或手动部署：
```bash
# 安装Vercel CLI
npm i -g vercel

# 部署
vercel
vercel --prod  # 部署到生产环境
```

### 优点：
- 永久免费（每月100GB流量）
- 自动HTTPS
- CDN加速
- 自定义域名支持

## 方案四：腾讯云/阿里云静态网站托管

### 腾讯云COS静态网站托管：
1. 访问：https://console.cloud.tencent.com/cos
2. 创建存储桶，选择"静态网站托管"
3. 上传网站文件
4. 启用默认域名或绑定自定义域名
5. 开启全球CDN加速

### 费用：约5-10元/月（基础套餐）

## 配置文件说明

### 1. ngrok-config.yml
```yaml
authtoken: YOUR_AUTH_TOKEN_HERE  # 从ngrok控制台获取
tunnels:
  moran-script:
    proto: http
    addr: 8899
    hostname: moran-script.ngrok-free.app
    region: ap
```

### 2. CORS配置（如需API访问）
创建 `C:\Users\admin\WorkBuddy\20260401001834\cors.json`：
```json
{
  "AllowedOrigins": ["*"],
  "AllowedMethods": ["GET", "POST"],
  "AllowedHeaders": ["*"]
}
```
