### 🔧 Vercel绑定域名步骤

#### 前提：已在Vercel部署网站

**步骤1：购买域名**
1. 访问阿里云/腾讯云
2. 搜索心仪域名
3. 完成购买

**步骤2：在Vercel绑定**
1. 登录Vercel控制台
2. 选择你的项目
3. 点击 "Settings" → "Domains"
4. 输入你的域名（如：moranscript.cn）
5. 点击 "Add"

**步骤3：配置DNS**
Vercel会显示需要添加的DNS记录：
```
类型   名称    值
CNAME  www     cname.vercel-dns.com
A       @      76.76.21.21
```

在域名平台添加这些记录：
- 阿里云：控制台 → 域名 → 解析设置
- 腾讯云：控制台 → 域名管理 → 解析

**步骤4：等待生效**
- DNS更新需要0-48小时
- Vercel自动申请SSL证书
- 完成后即可通过域名访问

### ⚡ 即时生效技巧

1. **修改本地hosts文件测试**：
```
# Windows: C:\Windows\System32\drivers\etc\hosts
76.76.21.21  moranscript.cn
76.76.21.21  www.moranscript.cn
```

2. **刷新DNS缓存**：
```bash
# Windows
ipconfig /flushdns

# Mac/Linux
sudo dscacheutil -flushcache
```

### 🆘 常见问题

**Q1：域名不生效怎么办？**
A：检查DNS配置，等待0-48小时，使用 ping 或 nslookup 测试。

**Q2：SSL证书会自动申请吗？**
A：Vercel和腾讯云都会自动申请Let's Encrypt免费SSL证书。

**Q3：网站搬家困难吗？**
A：不困难，Vercel支持一键迁移，只需修改DNS。

**Q4：需要备案吗？**
A：.cn 域名在国内服务器需要备案，在Vercel（境外）不需要。

**Q5：如何续费？**
A：域名到期前30-60天会收到邮件提醒，登录平台续费即可。
