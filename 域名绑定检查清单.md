## 🔄 第五步：在域名平台配置DNS
### 阿里云配置步骤：
1. [ ] 登录阿里云控制台：https://homenew.console.aliyun.com/
2. [ ] 选择 "域名"
3. [ ] 点击 "解析设置"
4. [ ] 添加CNAME记录：
   - 主机记录：www
   - 记录类型：CNAME
   - 记录值：cname.vercel-dns.com
   - TTL：600秒
5. [ ] 添加A记录：
   - 主机记录：@
   - 记录类型：A
   - 记录值：76.76.21.21
   - TTL：600秒

### 腾讯云配置步骤：
1. [ ] 登录腾讯云控制台：https://console.cloud.tencent.com/
2. [ ] 选择 "域名管理"
3. [ ] 点击 "解析"
4. [ ] 添加A记录：
   - 主机记录：@
   - 记录类型：A
   - 记录值：76.76.21.21
   - TTL：600
5. [ ] 添加CNAME记录：
   - 主机记录：www
   - 记录类型：CNAME
   - 记录值：cname.vercel-dns.com
   - TTL：600

## 🔍 第八步：常见问题排查
### 如果域名不生效：
1. [ ] **检查DNS配置**：确保记录值完全正确
2. [ ] **使用nslookup测试**：
   ```powershell
   nslookup www.你的域名
   nslookup 你的域名
   ```
3. [ ] **清空DNS缓存**：
   ```powershell
   ipconfig /flushdns
   ```
