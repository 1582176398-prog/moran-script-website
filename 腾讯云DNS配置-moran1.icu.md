# 腾讯云域名解析配置指南
## 域名：moran1.icu

## 🌐 必填DNS记录（共2条）

### 记录1：A记录（主域名）
- **作用**：让 moran1.icu 访问你的网站
- **配置参数**：
  ```
  主机记录：@
  记录类型：A
  记录值：76.76.21.21
  线路类型：默认
  权重：默认（可不填）
  TTL：600
  ```

### 记录2：CNAME记录（www子域名）
- **作用**：让 www.moran1.icu 也访问你的网站
- **配置参数**：
  ```
  主机记录：www
  记录类型：CNAME
  记录值：cname.vercel-dns.com
  线路类型：默认
  权重：默认（可不填）
  TTL：600
  ```

## 🔧 腾讯云操作步骤（截图版）

### 步骤1：登录腾讯云控制台
1. 访问：https://console.cloud.tencent.com/
2. 登录你的腾讯云账号

### 步骤2：进入域名管理
1. 搜索栏输入"域名"或直接访问：https://console.cloud.tencent.com/domain
2. 在域名列表中点击 `moran1.icu`
3. 点击 **"解析"** 按钮

### 步骤3：添加A记录（主域名）
1. 点击 **"添加记录"**
2. 按以下配置填写：
   - **主机记录**：`@`
   - **记录类型**：`A`
   - **记录值**：`76.76.21.21`
   - **线路类型**：`默认`
   - **TTL**：`600`
3. 点击 **"确定"**

### 步骤4：添加CNAME记录（www子域名）
1. 再次点击 **"添加记录"**
2. 按以下配置填写：
   - **主机记录**：`www`
   - **记录类型**：`CNAME`
   - **记录值**：`cname.vercel-dns.com`
   - **线路类型**：`默认`
   - **TTL**：`600`
3. 点击 **"确定"**

## ✅ 配置完成后检查

## 🐞 常见问题解决

### Q1：提示"主机记录已存在"
A：先删除或暂停同名记录，再添加新记录

### Q2：TTL值不可选600
A：选择"10分钟"或"600秒"，不同界面显示不同

### Q3：线路类型没有"默认"选项
A：选择"全网默认"或直接留空

### Q4：权重应该填什么？
A：留空或不填，保持默认即可

## 🧪 测试配置是否生效

### 方法1：Ping测试（立刻生效）
```powershell
ping moran1.icu
```
应该看到IP地址：`76.76.21.21`

### 方法2：nslookup测试（更准确）
```powershell
nslookup moran1.icu
nslookup www.moran1.icu
```
- moran1.icu 应返回：`76.76.21.21`
- www.moran1.icu 应返回：`cname.vercel-dns.com`

### 方法3：清理本地DNS缓存
```powershell
ipconfig /flushdns
```
